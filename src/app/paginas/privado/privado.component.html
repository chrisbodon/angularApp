Zona privada

Solo para usuarios logueados. Para tener acceso a esta ruta, debemos usar una guard y definirla en el routing.

path: 'privado', component: PrivadoComponent, canActivate: [LoginGuard]
canActivate
Guarda para indicar si se puede activar esa ruta.
LoginGuard
Guarda creada por nosotros, retorna true o false.
